# 长文档性能测试

本文档包含 500+ 行内容，用于测试 md2wechat 的渲染性能和编辑流畅度。

## 第一章：项目背景

在当今数字化时代，内容创作者面临着多平台发布的挑战。微信公众号作为中国最大的内容平台之一，其编辑器对外部样式的支持有限，给技术类文章的排版带来了极大困扰。

### 1.1 问题描述

微信公众号编辑器存在以下限制：

- 不支持外部 CSS 样式表
- 不支持 class 和 id 选择器
- 代码块的渲染效果不理想
- 表格样式需要手动调整
- 图片需要单独上传

### 1.2 解决方案

为了解决这些问题，我们开发了一个 Markdown 到公众号排版的转换工具。
